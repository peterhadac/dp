## CODING GUIDE (for data entry in MS Excel)

### General principles
- All Likert items: enter numeric value 1–5 directly
- All bipolar ordinal summary items (C7, D7, D11, D14): code 1–5 left to right
- Reverse-scored items (C3, C8, D4): recode as 6 − raw score before computing composites
- Multiple-select items (B1, B6, B8): enter as binary dummy variables per option (0/1)
- Missing / "prefer not to answer" / skipped: code as blank (leave cell empty); do not code as 0

### Dichotomisation for Chi-square tests

**H1 — B2 (frequency) × C7 (efficiency):**
- Frequency (B2): collapse to Low (1–3) vs. High (4–5)
- Efficiency (C7): collapse to Decreases/No change (1–3) vs. Increases (4–5)

**H2 — A3 (education) × B2 (frequency):**
- Education (A3): Secondary (option 1) vs. Higher education (options 2–5)
- Frequency (B2): Low (1–3) vs. High (4–5)

**H3 — A4 (experience) × D7 (attitude):**
- Experience (A4): 4 ordinal bands as coded (collapse if expected cell < 5)
- Attitude (D7): Negative (1–2) vs. Neutral (3) vs. Positive (4–5)

**H4 — D11 (EHR rating):**
- Positive = codes 4–5 ("somewhat improves" / "markedly improves")
- Run one-sample z-test: H0: p ≤ 0.50; H1: p > 0.50 (one-tailed, α = 0.05)
- Exclude respondents who selected "Never" at B3 from this analysis

### Composite scale scores (for descriptive statistics and Cronbach's alpha)
- **Efficiency composite (C-scale):** mean of C1 + C2 + (6−C3) + C5 + (6−C8) + C9 (6 items)
- **Attitude composite (D-scale):** mean of D1 + D2 + D3 + (6−D4) + D5 + D6 + D7 (7 items)
- **EHR impact composite:** mean of D8 + D9 + D10 + D11 + D12 (5 items)
- Compute Cronbach's alpha for each composite post hoc; report if α ≥ 0.70
